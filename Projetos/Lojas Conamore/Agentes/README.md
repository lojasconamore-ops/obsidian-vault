# Agentes | Lojas Conamore — Guia rápido

Este diretório organiza os agentes da Conamore em um padrão simples:

- **1 hub central**: `Agentes/Index.md`
- **1 índice por agente**: `Agentes/[Nome]/Index.md`
- **documentação detalhada**: fica no `SOUL.md` do perfil Hermes quando já existir material rico lá
- **treinamentos canônicos**: ficam nos docs da Conamore e são linkados pelos índices

## Como usar

- Comece pelo [hub de agentes](./Index.md)
- Abra o índice do agente que você quer consultar
- Use os links canônicos para chegar no treinamento certo

## Regra de manutenção

Evite duplicar o mesmo conteúdo completo em vários lugares. Prefira criar links e manter um único ponto canônico por assunto.
