# 🏔️ Bem-vindo ao seu Vault Obsidian

Este vault está configurado no servidor **agentic-vm**.

## Como usar

Crie notas em Markdown (`.md`) em qualquer subpasta. O Hermes pode:

- **Criar notas** com `write_file`
- **Ler notas** com `read_file`
- **Pesquisar** com `search_files`
- **Editar** com `patch`
