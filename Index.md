# 🏔️ Bem-vindo ao seu Vault Obsidian

Este vault está configurado no servidor **agentic-vm**.

## Como usar

Crie notas em Markdown (`.md`) em qualquer subpasta. O Hermes pode:

- **Criar notas** com `write_file`
- **Ler notas** com `read_file`
- **Pesquisar** com `search_files`
- **Editar** com `patch`

## Estrutura sugerida

```
Documents/Obsidian Vault/
├── Index.md                 ← Você está aqui
├── Projetos/               ← Notas de projetos
├── Diário/                 ← Notas diárias
└── Referências/            ← Documentação, links
```

## Links

Use `[[Nome da Nota]]` para criar links entre notas no estilo Obsidian.
