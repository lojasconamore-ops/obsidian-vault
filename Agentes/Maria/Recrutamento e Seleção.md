# Recrutamento e Seleção

## Perfis Recorrentes

| Perfil | Departamento | Características |
|--------|-------------|-----------------|
| **Costureira** | Fábrica | Experiência em costura industrial, máquinas reta e overloque |
| **Auxiliar de Produção** | Fábrica | Corte, acabamento, embalagem |
| **Vendedor(a)** | Lojas / Comercial | Experiência em varejo ou vendas B2B |
| **Analista Administrativo** | Admin | Financeiro, compras, RH, marketing |
| **Motorista / Logística** | Logística | Entrega, expedição, cargas |

## Etapas do Processo

1. **Triagem de currículos** — plataformas (Indeed, LinkedIn, banco interno)
2. **Entrevista telefônica** — fit inicial, disponibilidade, pretensão
3. **Entrevista presencial** — com gestor da área
4. **Prova prática** (quando aplicável) — costura, vendas, informática
5. **Oferta** — proposta salarial, benefícios, data de início
6. **Admissão** — documentação, exame médico, registro

## Documentos para Admissão

- [ ] Carteira de Trabalho (CTPS digital)
- [ ] RG e CPF
- [ ] Comprovante de residência
- [ ] Certidão de casamento (se aplicável)
- [ ] Certidão de nascimento dos filhos (para salário-família)
- [ ] Comprovante de escolaridade
- [ ] PIS/PASEP
- [ ] Título de eleitor
- [ ] Reservista (para homens)
- [ ] Exame médico admissional
- [ ] Contrato de trabalho (2 vias)
- [ ] Termo de ciência de políticas internas
