# LGPD para RH — Dados de Colaboradores

## Dados Pessoais Tratados pelo RH

| Categoria | Dados |
|-----------|-------|
| **Identificação** | Nome, RG, CPF, CTPS, PIS, título de eleitor |
| **Contato** | Endereço, telefone, e-mail |
| **Familiares** | Cônjuge, filhos (para salário-família, dependentes IR) |
| **Financeiros** | Salário, comissões, benefícios, banco para pagamento |
| **Saúde** | Atestados, exames admissionais/demissionais/periódicos |
| **Biométricos** | Foto, assinatura, impressão digital (ponto) |
| **Disciplinares** | Advertências, suspensões, avaliações de desempenho |

## Bases Legais (LGPD Art. 7º)

| Finalidade | Base Legal |
|------------|-----------|
| Execução do contrato de trabalho | Art. 7º, V |
| Obrigação legal (CLT, previdência, FGTS) | Art. 7º, II |
| Medicina do trabalho | Art. 7º, II + Art. 11, II (dado sensível) |
| Proteção ao crédito (consulta SPC/Serasa) | Art. 7º, X |
| Legítimo interesse (avaliação, promoção) | Art. 7º, IX |

## Cuidados Específicos do RH

- **Dados de saúde** são sensíveis — acesso restrito e controlado
- **Atestados médicos** não podem ser compartilhados sem necessidade
- **Exames** admissionais, demissionais e periódicos — guarda sigilosa
- **Dados bancários** — apenas para folha de pagamento
- **Informações disciplinares** — acesso apenas a gestores e DP

## Direitos dos Titulares (Colaboradores)

- Solicitar acesso aos seus dados
- Corrigir dados incorretos
- Solicitar eliminação de dados desnecessários (após obrigação legal)
- Revogar consentimento (quando aplicável)

## Política de Retenção

| Documento | Prazo de Guarda |
|-----------|:---------------:|
| Contrato de trabalho | 5 anos após rescisão |
| Folha de pagamento | 30 anos |
| GFIP/SEFIP | 30 anos |
| Exames médicos | 20 anos |
| Atestados médicos | 5 anos |
| Ponto eletrônico | 5 anos |

## Checklist de Conformidade RH

- [ ] Política de privacidade para colaboradores divulgada
- [ ] Termo de consentimento para uso de imagem (se aplicável)
- [ ] Controle de acesso a sistemas de RH (quem vê o quê)
- [ ] Procedimento para resposta a solicitações de titulares
- [ ] Registro de tratamento de dados de colaboradores
- [ ] Contratos com cláusulas LGPD (benefícios, planos de saúde)
