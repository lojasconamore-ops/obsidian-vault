# Stack Tecnológica da Conamore

## Infraestrutura

| Sistema | Função | Detalhes |
|---------|--------|----------|
| **Magento/SignaCommerce** | E-commerce | Loja online conamore.com.br |
| **RD Station** | CRM/Marketing | 27 tools, APIs |
| **Intelipost** | TMS Logística | API REST de fretes |
| **SQL Server** | Banco de dados | hotel-finder e outros |
| **Composio** | Integrações Google | Gmail, Drive, Docs, Analytics, Search Console |
| **Linux Server** | Servidor principal | Azure VM |

## Responsabilidades Técnicas

- Disponibilidade do e-commerce (Magento)
- Integrações entre sistemas (API Intelipost, RD Station, Magento)
- Backup e recuperação de dados SQL Server
- Segurança da informação e LGPD (camada técnica)
- Suporte a usuários internos
- Monitoramento de performance e uptime
